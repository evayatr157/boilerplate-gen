# FastAPI Prisma Starter Kit

## Setup

1. Clone the repository and navigate into the project directory.
2. Run the setup script:
   ```bash
   npm install
   npm run setup
   ```
3. Provide the required environment variables when prompted.

## Running the Application

1. Start the application using Docker Compose:
   ```bash
   docker-compose up
   ```
2. Access the FastAPI application at [http://localhost:8000](http://localhost:8000).

The PostgreSQL database is set up using Docker and Prisma is used for the ORM.
